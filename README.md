## 📦 Instalación

### 1️⃣ Clonar la app**

```bash
git clone https://github.com/tu-usuario/tu-proyecto.git
```

### 2️⃣ Iniciar la app**

```bash
cd tu-proyecto
npm install
ng serve y navegar a `http://localhost:4200/` o también puedes poner directamente *ng serve -o*
```
